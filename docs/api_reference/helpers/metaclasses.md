# Metaclasses

The `helpers/metaclasses` subpackage provides Python metaclass implementations for structural patterns such as the singleton, giving classes framework-level behavioral guarantees.

## Singleton

A thread-safe metaclass implementation of the singleton pattern that ensures a class has exactly one instance.
